## Supported operating systems
* Linux
* MacOS
* Windows

## Build requirements
* C99 compiler
* [CMake 3.10+](https://cmake.org/)
* [X11](https://www.x.org/) (Linux only)

## X11 installation
* Linux: sudo apt install xorg-dev
